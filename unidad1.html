¿Qué es un sistema numerico?

Cualquier sistema consta de una serie de elementos que lo conforman una serie de reglas que permite establecer operaciones y relaciones entre tales elementos. Por ello, puede decirse que un sistema de numeración es el conjunto de elementos (símbolos o números), operaciones y relaciones lo conforman

Sistema Analógico

en la ciencia, tecnología, administración, etc. se manejan constantemente a las cantidades, estas se miden monitorean, registran, manipulan, aritméticamente, observan o en alguna otra forma se utilizan en muchos sistemas físicos. Cuando se manejan diversas cantidades, es importante que podamos representar sus valores con eficiencia y exactitud, puede haber valores numéricos, analógicos y digitales en la representación analógica se presenta un voltaje corriente o movimiento de un indicador o un medidor que es proporcional al valor de esa cantidad, Las cantidades analógicas tienen una característica importante, Las cantidades no se representan por valores proporcionales, sino por símbolos, denominados dígitos, el sistema Analógico está en continuo cambio, una cantidad analógica está abierta a la interpretación

Sistema Digital

el sistema digital es el que más se usa en los Sistemas decimal, binario, octal y Hexadecimal, Las características del sistema digital es que es más fácil de transmitir, procesar y almacenar esa información que es más exacto y preciso que el analógico y que tiene mayor estabilidad

Sistema Binario

sólo hay dos símbolos o posibles valores de dígitos 0:01. No obstante, este sistema de base dos se puede utilizar para representar cualquier cantidad que se requiera en un sistema decimal o algún otro sistema numérico. En general, se necesitarán muchos más dígitos, binarios para expresar una cantidad determinado. Esto es un sistema de valor proporcional en el sistema binario. El término dígito binario se abrevia a menudo como Bit para determinar su equivalente en el sistema decimal, simplemente tomamos la suma de los productos de cada valor y su valor posicional

Sistema Decimal

el sistema de numeración más utilizado es el sistema Numérico decimal que tiene una base de 10 dígitos Usa 10, símbolos para representar los valores numéricos que son los dígitos del cero al nueve, es un sistema dependiente de orden, el valor numérico se obtiene sumando los productos de cada dígito por la base 10 elevada a la posición que ocupa ese dígito

Conversion Binario a Decimal

cada dígito en un número binario compuesto sólo de uno y cero representa una potencia de 2 desde la derecha hacia la izquierda

Por ejemplo: 1101 a decimal

Como se resolvio anteriormente, el 1 al inicio se multiplico por 2 elevado a 3 debido a que esta en base 2 y se elevo a la 3 porque son 4 digitos y es el que esta más a la derecha, luego le seguía otro 1 y se elevo a la 2 siguiendo el patrón de derecha a izquierda, luego el 0 se multiplico con 2 elevado a 1, y el último 1 se multiplico con 2 a la 0 ya que es el que esta mas a la izquierda
